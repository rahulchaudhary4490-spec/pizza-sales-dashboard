# 🍕 Pizza Sales Dashboard (SQL + Excel)

## 📌 Project Overview
This project analyzes pizza sales data using SQL and visualizes insights using an Excel dashboard.

## 🛠 Tools Used
- SQL (for data analysis)
- Excel (for dashboard & visualization)

## 📊 Key Insights
- Total Revenue: ₹64,180
- Avg Order Value: ₹38.64
- Total Orders: 1661
- Busiest Days: Friday & Saturday evenings
- Most Sold Category: Classic
